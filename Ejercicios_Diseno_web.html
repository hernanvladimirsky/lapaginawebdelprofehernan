Ejercicios Introductorios

Ejercicio Introductorio 1

En base a un tema musical que me guste, crear una página web.
Respetando la estructura básica de html y con sus respectivas etiquetas insertar:

  1. Insertar un título.
  2. Insertar un parrafo.
  3. Insertar una imagen.

Ejercicio Introductorio 2

A partir de un receta de cocina que me guste, describir que ingredientes necesita el plato y sus pasos a seguir para su elaboracion.

  1. Insertar lista ordenada.
  2. Insertar lista desordenada.
  3. Insertar lista descriptiva.

Ejercicio Introductorio 3

Redactar un email a una institucion para averiguar sobre una carrera universitaria, utilizar negrita, cursiva y subrayado en las palabras que considere relevantes.

Ejercicio Introductorio 4

Crear un vinculo en la pagina principal que me redirija a la pagina secundaria, esta ultima se debe abrir en otra pestaña.

Ejercicio Introductorio 5

Crear una imagen en paint, insertarla en mi pagina web y utilizar los atributos, title, alt, widht y height.

Ejercicio Introductorio 6

Cada elemento semantico debe tener un color de fondo y su respectivo nombre.

Trabajo Práctico Integrador - Crear mi Empresa

Para comenzar; remarco que es una actividad exclusivamente creativa donde no hay margenes preestablecidos, solo utilizar la propia imaginación.

Crear una empresa; para ello crear una página web principal que muestra al cliente quienes somos y a que nos dedicamos.
Luego crear distintas paginas secundarias para mostrar el catalogo de nuestros productos y para que el cliente se contacte con nosotros.

Herramientas a utilizar:

  • Estructura básica de una página web.
  • Elementos semánticos.
  • Listas ordenadas y desordenadas.
  • Tablas.
  • Insertar vinculos e imagenes.
  • Crear formularios.
  • ID y clase.
  • Anclas.
  • Aplicar diferentes tipos de CSS a elección.
  • Propiedad Margin.
  • Propiedad Padding.
  • Propiedad Float.

Ejercicios

Ejercicio 1

  1. Crear un documento html.
  2. Crear un documento css.
  3. Enlazar ambos aplicando Css Externo.
  4. Insertar una imagen a eleccion y alinearla al centro de su contenedor con la propiedad margin-left o margin-right.
  5. Insertar un párrafo, aplicarle color de letra, color de fondo, tamaño de letra y text-align: center (luego probar con el valor right y left en vez de center).
  6. Aplicar la propiedad padding al parrafo (probar cada tipo de padding).
¿Que hace la propiedad Margin y Padding? Escribir la respuesta en la carpeta.

Ejercicio 2

  1. Crear un documento html con 3 etiquetas p y aplicar css en linea.
  2. Crear tres documentos html con 3 etiquetas p y aplicar css interno.
  3. Crear tres documentos html con una etiqueta p y aplicar css externo.
Para aplicar Css externo recordar crear el archivo estilos.css y enlazarlo con los archivos html, ambos deben estar en la misma carpeta.
El contenido de cada etiqueta debe ser "soy un parrafo" y pueden poner el color de letra, color de fondo, tamaño de letra a elección.

¿En que se diferencian Css Externo, Interno y En linea?
¿Cuando es recomendable utilizar cada uno?
Escribir las respuestas en la carpeta.

Ejercicio 3

  1. Crear un documento html y una hoja de estilo aplicando css externo.
  2. Crear 4 titulos de nivel 1 (el de mayor importancia para los moteres de busqueda).
  3. Tres titulos deben pertenecer a una misma clase, aplicar un estilo a elección para ellos.
  4. El restante debe tener un identificador único tambien aplicarle un estilo a elección.

Ejercicio 4

Aplicando Css externo con los contenidos vistos en ejercicios anteriores, realizar el siguiente ejercicio:

  1. Insertar una imagen y alinearla hacia margen derecho de la pantalla.
  2. Insertar un parrafo y alinear el contenido hacia abajo, utilizar un color de fondo.
  3. Crear otro parrafo, separar el contenedor del contenido utilzando el mismo valor para los cuatro lados, utilizar un color de fondo.

Ejercicio 5

  1. Crear un título en negrita, aplicarle un color a elección y un tamaño de letra pequeño.
  2. Crear un parrafo y alinear el contenido hacia el centro.
  3. Crear un titulo y un parrafo aplicarles un color de fondo, separar ambos elementos.

Ejercicio 6

  1. Insertar una imagen y un párrafo con un contenido a elección, relacionando el parrafo con la imagen.
  2. El contenido del párrafo debe estar en el centro, y debe tener un color de fondo.
  3. Ubicar el parrafo a la derecha de la imagen, utilizando la propiedad float.
  4. Insertar una lista ordenada con un contenido a elección.

Ejercicio 7

  1. Insertar un parrafo, y una imagen.
  2. Alinear el texto del parrafo a la derecha.
  3. Separar el contenedor del contenido, quedando el contenido del parrafo debajo.
  4. Ubicar la imagen a la derecha y el parrafo a la izquierda.
  5. Debajo insertar un link que me envie a una pagina secundaria llamada contacto.html
  6. Separar el link de los elementos de arriba.
