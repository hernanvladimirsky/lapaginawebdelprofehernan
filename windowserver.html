🌐 Configuración de Adaptador de Red en VirtualBox

🧭 1. Ir a la configuración de red

  1. Abrir VirtualBox
  2. Seleccionar la máquina virtual (sin iniciar)
  3. Hacer clic en Configuración
  4. Ir a la pestaña Red

🔌 2. Elegir el tipo de adaptador

🌍 NAT

  • Usa la conexión de tu PC
  • Tiene internet automáticamente
  • No es visible en la red

🖧 Adaptador Puente (Bridge)

  • La VM actúa como una PC más
  • Recibe IP del router (DHCP)
  • Puede comunicarse con otras máquinas

🔒 Red Interna

  • Solo comunicación entre máquinas virtuales
  • No tiene internet

💻 Solo-Anfitrión (Host-Only)

  • Comunicación entre tu PC y la VM
  • Sin internet (por defecto)

⚙️ 3. Configurar detalles

  • Bridge: Elegir placa de red (WiFi o Ethernet)
  • Host-Only: DHCP automático o IP manual
  • NAT: Generalmente no requiere cambios

💻 4. Configuración dentro del sistema operativo

🪟 Windows

  • Panel de control → Red
  • Configurar IP automática (DHCP) o manual

🐧 Linux

  • Comando: ip a
  • Configurar desde Network Manager

💡 Tip para prácticas de redes

Usar 2 adaptadores:
  • 🌍 Adaptador 1: NAT (internet)
  • 🔒 Adaptador 2: Host-Only (laboratorio)
Permite hacer prácticas con DHCP, servidores y clientes.

🖥️ Ejercicio 1: Configuración Básica de Windows Server

🎯 Objetivo

Configurar parámetros básicos del servidor: nombre, escritorio remoto, red (DHCP y estática) y verificar configuración con comandos.

✏️ 1. Cambiar el nombre del servidor

  1. Ir a Inicio
  2. Click derecho en Este equipo → Propiedades
  3. Seleccionar Cambiar configuración
  4. Click en Cambiar
  5. Ingresar nuevo nombre (Ej: SERVER-01)
  6. 🔁 Reiniciar el equipo

🖥️ 2. Activar Escritorio Remoto

  1. Ir a Configuración
  2. Entrar en Sistema
  3. Seleccionar Escritorio remoto
  4. Activar la opción
  5. Permitir usuarios si es necesario

🌐 3. Configurar IP por DHCP

  1. Ir a Panel de control
  2. Red e Internet → Centro de redes
  3. Cambiarle el nombre por Internet
  4. Cambiar configuración del adaptador
  5. Click derecho en la red → Propiedades
  6. Seleccionar Protocolo IPv4
  7. Elegir:
    • ✔ Obtener IP automáticamente
    • ✔ Obtener DNS automáticamente

📌 4. Configurar IP Estática

  1. Cambiarle el nombre por LAN
  2. Ir nuevamente a IPv4
  3. Seleccionar Usar la siguiente dirección IP
  4. Completar:
    • IP: 192.168.1.10
    • Máscara: 255.255.255.0
    • Puerta de enlace: 192.168.1.1
    • DNS: 8.8.8.8

🔍 5. Verificar configuración con ipconfig

  1. Abrir CMD
  2. Ejecutar:

ipconfig

También se puede usar:

ipconfig /all

✅ Resultado esperado

  • Servidor con nombre configurado
  • Escritorio remoto habilitado
  • Red funcionando en DHCP
  • Red funcionando con IP estática
  • Verificación correcta con ipconfig

🏢 Ejercicio 2: Instalación de Active Directory

📚 ¿Qué es Active Directory?

Active Directory (AD) es un servicio que permite administrar y centralizar todos los recursos de una red desde un servidor.

Funciona como una base de datos donde se almacenan usuarios, computadoras, contraseñas y permisos.

🛠️ ¿Para qué sirve?

  • Permite iniciar sesión en cualquier PC con la misma cuenta
  • Centraliza la administración de la red
  • Aplica políticas de seguridad
  • Organiza la red mediante un dominio

🏢 Ejemplo de empresa

La empresa tiene dos sucursales:

  • 📍 Ramos Mejía
  • 📍 San Justo

Dominio a crear:

empresa.com

🎯 Objetivo

Instalar Active Directory y crear el dominio empresa.com

⚙️ 1. Abrir el Administrador del Servidor

  1. Abrir Administrador del servidor
  2. Click en Agregar roles y características

⚙️ 2. Tipo de instalación

  1. Seleccionar Instalación basada en características o roles
  2. Click en Siguiente

⚙️ 3. Seleccionar servidor

  1. Elegir el servidor
  2. Click en Siguiente

⚙️ 4. Seleccionar rol

  1. Activar Servicios de dominio de Active Directory
  2. Aceptar agregar características
  3. Click en Siguiente

⚙️ 5. Instalar

  1. Click en Instalar
  2. Esperar que finalice

🏗️ 6. Promover a controlador de dominio

  1. Click en la bandera amarilla
  2. Seleccionar Promover este servidor a controlador de dominio
  3. Elegir Agregar un nuevo bosque
  4. Ingresar dominio: empresa.com
  5. Configurar contraseña
  6. Continuar con Siguiente hasta instalar

🔁 7. Reinicio del servidor (CLAVE)

  1. El sistema pedirá reiniciar
  2. Aceptar reinicio
  3. Esperar a que el servidor reinicie

Importante: Sin reiniciar, Active Directory no funcionará correctamente.

🔍 8. Verificación

  • Iniciar sesión como: EMPRESA\Administrador
  • Verificar funcionamiento del dominio

✅ Resultado esperado

  • Active Directory instalado
  • Dominio empresa.com funcionando
  • Servidor como controlador de dominio
  • Base para organizar sucursales (Ramos Mejía y San Justo)
