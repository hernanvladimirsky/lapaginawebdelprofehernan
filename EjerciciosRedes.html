Trabajo practico 1 - Fundamentos de redes

Crear un documento Word, insertar portada para el TP y responder las siguientes preguntas insertando imagenes.

  1. ¿Cuales son las razones para instalar redes?
  2. Clasificar las redes segun la utilizacion de los usuarios.
  3. Clasificar las redes segun los servicios que brindan al usuario.
  4. Clasificar las redes segun la propiedad a la que pertenecen.
  5. ¿Cuales son los tipos de redes segun su tamaño(cobertura del servicio)?
  6. ¿Que es un gateway y como lo describirias?
  7. ¿Cuales es la clasificación de las redes según la tecnología de transmisión?
  8. Clasificar las redes según el tipo de transferencia de datos que soportan.
  9. ¿Que tipos de medios y formas para la transmisión de datos existen?
  10. Clasificar las redes segun su uso.
  11. ¿Que tipologias de red existen? Insertar imagenes de ellas.
  12. Realizar en Cisco Packet Tracer la topologia de red bus, estrella y arbol. Cada una de ellas deber tener 5 (cinco) host.
  13. ¿Que es una placa de red y cual es su función?
  14. Explicar en que consiste la preparacion de datos, direcciones de red, envio y control de datos.

Trabajo practico 2 - IP ,Redes

Crear un documento Word, insertar portada para el TP y responder las siguientes preguntas:

  1. ¿Qué es una dirección IP?
  2. ¿Qué es una dirección IP privada y para que puede servir?
  3. ¿Qué es una dirección IP pública y para que puede servir?
  4. Realiza una tabla de la Clase A, Clase B y Clase C (con su rango, mascara de red, host y uso) y nombrar para que sirve cada una.
  5. ¿Qué significa /24 ,/16 y /8 en una dirección IP? Explicar cada una.
  6. ¿Qué es la máscara de red? ¿Para que sirve?
  7. ¿En que se diferencian las direcciones IP IPv4 e IPv6? ¿Con que objetivo se hizo IPv6?
  8. ¿Que diferencias hay entre una dirección IP y una dirreccion MAC?
  9. ¿Como funcionan los routers?
