Breve historia

El lenguaje C++ fue creado por Bjarne Stroustrup a comienzos de la década de 1980 en los laboratorios Bell Labs (Estados Unidos). Stroustrup buscaba un lenguaje que combinara la eficiencia y el control de C con las características de programación orientada a objetos que ofrecían otros lenguajes de la época, como Simula.

En un principio, C++ se llamó “C with Classes” (C con clases), ya que era una extensión del lenguaje C que añadía la posibilidad de definir clases, objetos, herencia y encapsulamiento.

En 1983, el lenguaje adoptó el nombre C++, haciendo referencia al operador de incremento (++) del propio lenguaje, que simboliza una versión mejorada de C.

Durante los años siguientes, C++ se consolidó como uno de los lenguajes más usados en la industria del software gracias a su velocidad, portabilidad y flexibilidad. Fue normalizado por primera vez por ISO en 1998 (C++98), con revisiones importantes posteriores:

  • C++03 (2003): correcciones menores.
  • C++11 (2011): nuevas características como auto, nullptr y lambdas.
  • C++14, C++17, C++20 y C++23: incorporan mejoras de rendimiento y características modernas.

Hoy, C++ sigue siendo uno de los lenguajes más utilizados en el desarrollo de sistemas operativos, videojuegos, motores gráficos, software científico e inteligencia artificial.

🧩 Estructura básica de un programa en C++

Un programa en C++ tiene una estructura general que se puede dividir en varias partes fundamentales:

Parte Descripción
1. Directivas de preprocesador Indican al compilador que incluya librerías necesarias, como <iostream> para entrada y salida de datos.
2. Función principal (main()) Es el punto de inicio del programa. Todo programa C++ debe tener una función main().
3. Declaración de variables Espacio donde se definen variables que almacenarán datos.
4. Instrucciones o sentencias Son las órdenes que el programa ejecutará, como mostrar mensajes o leer datos.
5. Retorno de valor La función main() devuelve un valor al sistema operativo (normalmente return 0;).

📘 Ejemplo de estructura básica (comentado)

// Ejemplo básico de programa en C++
// Autor: [Tu nombre]
// Descripción: muestra un mensaje por pantalla

#include <iostream>     // Librería para entrada y salida de datos
using namespace std;    // Permite usar cout y cin sin escribir std::

int main() {
    // Declaración de variables
    int edad;                        // Variable para almacenar edad
    string nombre;                   // Variable para almacenar nombre

    // Entrada de datos
    cout << "Ingrese su nombre: ";   // Muestra mensaje al usuario (salida)
    cin >> nombre;                   // Lee el nombre desde el teclado (entrada)

    cout << "Ingrese su edad: ";     // Solicita la edad
    cin >> edad;                     // Lee la edad desde el teclado

    // Salida de datos
    cout << "Hola " << nombre << ", tienes " << edad << " años." << endl;

    // Fin del programa
    return 0;   // Retorna 0 indicando que el programa terminó correctamente
}

🧠 Explicación paso a paso

  • #include <iostream>: Incluye la biblioteca estándar de entrada/salida (Input/Output Stream).
  • using namespace std;: Permite omitir el prefijo std:: en los objetos estándar (cout, cin, string).
  • int main(): Es la función principal. Todo programa en C++ comienza su ejecución desde aquí.
  • Declaraciones y sentencias: Dentro de { } se escribe el código que se ejecutará. Cada instrucción termina con punto y coma ;.
  • return 0;: Indica que el programa terminó correctamente.

💬 Explicación de cout, cin, << y >>

En C++, la comunicación con el usuario se realiza mediante entrada y salida de datos:

  • cout → Significa “character output” (salida de caracteres). Se usa para mostrar información en pantalla.
  • cin → Significa “character input” (entrada de caracteres). Se usa para leer datos que el usuario ingresa desde el teclado.

Los operadores especiales que acompañan a cout y cin son:

  • << → Operador de inserción. Envía información hacia la salida estándar (cout).
  • >> → Operador de extracción. Obtiene información desde la entrada estándar (cin).

Ejemplo:

cout << "Hola mundo";   // Muestra el texto en pantalla
cin >> nombre;         // Espera que el usuario escriba algo y lo guarda en la variable 'nombre'

1. Operadores de Comparación

Se utilizan para comparar valores dentro de estructuras como if, while y for.

Operador Significado Ejemplo
> Mayor que a > b
< Menor que a < b
>= Mayor o igual que a >= b
<= Menor o igual que a <= b
== Igual a a == b
!= Distinto de a != b

Ejemplo con comparación


int edad = 20;

if (edad >= 18) {
    cout << "Es mayor de edad";
}

2. Operadores Lógicos

Permiten combinar varias condiciones en una sola expresión.

Operador Nombre Significado Ejemplo
&& AND (Y) Verdadero si ambas condiciones se cumplen a > 0 && b > 0
|| OR (O) Verdadero si al menos una condición se cumple a > 0 || b > 0
! NOT (NO) Invierte el valor lógico !(a > b)

Ejemplo con AND


int edad = 30;

if (edad >= 18 && edad <= 65) {
    cout << "Edad dentro del rango laboral";
}

Ejemplo con OR


int nota = 4;
int asistencia = 85;

if (nota >= 6 || asistencia >= 80) {
    cout << "Puede rendir el examen final";
}

Ejemplo con NOT


int edad = 16;

if (!(edad >= 18)) {
    cout << "No es mayor de edad";
}

3. Uso de cin y cout

cout << se utiliza para mostrar información en pantalla.

cin >> se utiliza para ingresar datos desde el teclado.

Ejemplo completo


#include <iostream>
using namespace std;

int main() {

    int numero;

    cout << "Ingrese un numero: ";
    cin >> numero;

    if (numero > 0) {
        cout << "El numero es positivo";
    } else {
        cout << "El numero es negativo o cero";
    }

    return 0;
}

🔄 WHILE

El ciclo while es una estructura repetitiva que se ejecuta mientras una condición sea verdadera. Primero evalúa la condición y, si se cumple, ejecuta el bloque de código. Este proceso se repite hasta que la condición deja de cumplirse. 

  #include <iostream>
  using namespace std;
  
  int main() {
      int i = 1; // Variable contador que inicia en 1
  
      // El while se ejecuta mientras i sea menor o igual a 5
      while (i <= 5) {
          cout << "Es la iteracion: " << i << endl; // Muestra el valor de i
          i++; // Incrementa i en 1 para evitar un bucle infinito
      }
  
      return 0; // Fin del programa
  }

⏳ DO WHILE

El ciclo do while es una estructura repetitiva que ejecuta el bloque de código al menos una vez, y luego evalúa la condición. Si la condición es verdadera, el ciclo se repite. Este proceso continúa hasta que la condición deja de cumplirse. 

#include <iostream>
using namespace std;

int main() {
    int i = 1; // Variable contador que inicia en 1

    do {
        cout << "Es la iteracion: " << i << endl;
        i++; // Incrementa i
    } while (i <= 5);

    return 0;
}

⚙️ FOR

El ciclo for es una estructura repetitiva que se utiliza cuando se conoce la cantidad de veces que se quiere repetir un bloque de código. Tiene tres partes: inicialización, condición y actualización. El ciclo se ejecuta mientras la condición sea verdadera. 

#include <iostream>
using namespace std;

int main() {
  
  // Inicialización, condición y aumento en una sola línea
  for (int i = 1; i <= 5; i++) {
      cout << "Es la iteracion: " << i << endl;
  }

  return 0;
}

🔁 WHILE

Usalo cuando no sabés cuántas veces se va a repetir.
Primero evalúa la condición y puede no ejecutarse nunca.

⏳ DO WHILE

Usalo cuando querés que el código se ejecute al menos una vez.
Primero ejecuta y después evalúa la condición.

⚙️ FOR

Usalo cuando sabés cuántas veces se va a repetir.
Tiene inicialización, condición y aumento en una sola línea.

Resumen

  • for → contado
  • while → depende
  • do while → mínimo una vez
